- **Deep Learning**: BPTT with LSTM-style recurrent networks
- **Spiking Neural Networks**: Izhikevich neurons with STDP and temporal coding

- **Real-time Overlay**: FSM state, system metrics, API key, FPS

### ✅ **Security & Persistence**
- **Cryptographic Keys**: 256-bit API keys with 28-hour rotation
- **Thread Safety**: Mutex-protected shared data
- **Performance Monitoring**: FPS counter, computation cycles
